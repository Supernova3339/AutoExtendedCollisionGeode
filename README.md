<cr>**Don't you just wish that object hitboxes didn't break when scaling/warping them?</c>**



**<cg>Well look no further, this mod has the solution!</c>**



<cb>Auto Extended Collision</c> activates the <cg>"Extended Collision"</c> option everytime you scale/warp an object beyond 4x (You can customize the default scale in settings) so you don't have to go through the annoying process of going from menu to menu only to activate it!
